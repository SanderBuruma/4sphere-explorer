Create lib/compass.py. Structure:

1. Imports: pygame, numpy as np, math

2. Fixed standard basis constants (module-level, never modified):
```python
_AXIS_X = np.array([1.0, 0.0, 0.0, 0.0])
_AXIS_Y = np.array([0.0, 1.0, 0.0, 0.0])
_AXIS_Z = np.array([0.0, 0.0, 1.0, 0.0])
_AXIS_W = np.array([0.0, 0.0, 0.0, 1.0])
```

3. Lerp animation state (module-level):
```python
_needle_angle = 0.0
_target_angle = 0.0
_lerp_progress = 1.0   # start at completion so first frame snaps cleanly
_LERP_DURATION_MS = 200.0
```

4. Three calculation functions (public, testable in isolation):

`calculate_heading(camera_pos)` -- dot product with _AXIS_X and _AXIS_Z, use np.arctan2(-z_comp, x_comp). Returns float in [-pi, pi]. No clamping needed for arctan2.

`calculate_tilt(camera_pos)` -- dot product with _AXIS_Y, np.clip to [-1, 1], then np.arccos(abs(y_comp)). Returns float in [0, pi/2] (0 = aligned with Y, pi/2 = perpendicular).

`calculate_w_alignment(camera_pos)` -- dot product with _AXIS_W, np.clip to [-1, 1]. Returns float in [-1, 1].

5. `_update_needle(target_heading, dt_ms)` -- private, updates module-level Lerp state:
  - If abs difference between _needle_angle and target_heading > 0.001: set _target_angle = target_heading, _lerp_progress = 0.0
  - If _lerp_progress < 1.0: increment by dt_ms / _LERP_DURATION_MS, clamp to 1.0
  - Compute delta = _target_angle - _needle_angle; if delta > pi: subtract 2*pi; elif delta < -pi: add 2*pi
  - _needle_angle += delta * _lerp_progress (not divide by lerp_progress -- this is direct t-weighted step each frame)
  - Note: The lerp here is "ease toward target" style -- each frame moves lerp_progress fraction of remaining delta. When lerp_progress hits 1.0 snap _needle_angle = _target_angle.

6. `render_compass(screen, orientation, x, y, size=120)` -- main public function:
  a. Extract camera = orientation[0]
  b. Calculate heading, tilt, w_align
  c. Get dt_ms via pygame.time.Clock -- but since we have no clock reference, use a module-level _last_render_ms tracking. On first call: _last_render_ms = pygame.time.get_ticks(), dt_ms = 16. Subsequent calls: dt_ms = pygame.time.get_ticks() - _last_render_ms (clamped to [1, 100]), update _last_render_ms.
  d. Call _update_needle(heading, dt_ms)
  e. Create widget_surf = pygame.Surface((size, size), pygame.SRCALPHA)
  f. Draw semi-transparent background: pygame.draw.rect(widget_surf, (30, 30, 50, 180), (0, 0, size, size), border_radius=6)
  g. Draw subtle border: pygame.draw.rect(widget_surf, (100, 100, 130, 100), (0, 0, size, size), 1, border_radius=6)
  h. Compass rose area: center at (size*0.42, size*0.5), rose_radius = size * 0.30
     - Draw circle outline for rose: pygame.draw.circle(widget_surf, (60, 65, 90), center, rose_radius, 1)
     - Draw tick marks at 4 cardinal angles (0, pi/2, pi, 3pi/2)
     - Draw X+, X-, Z+, Z- labels at rose_radius + margin from center. Label angles: X+ at 0, Z+ at pi/2, X- at pi, Z- at 3*pi/2 (in screen coords: lx = cx + (rose_radius+offset)*cos(angle), ly = cy - (rose_radius+offset)*sin(angle)). Use pygame.font.Font(None, max(10, size//12)) for label font.
     - Draw compass rose circle ticks (4 small lines at cardinal positions)
  i. Draw needle from center to heading direction:
     - needle_len = rose_radius * 0.85
     - nx = cx + needle_len * math.cos(_needle_angle)
     - ny = cy - needle_len * math.sin(_needle_angle)
     - Draw line width 2, color (255, 110, 80) (orange-red)
     - Draw small filled circle at needle tip radius 3
     - Draw short back stub (opposite direction, 1/4 length) in darker color (120, 60, 40) to visually indicate direction
  j. Tilt bar (right side of widget):
     - bar_cx = size * 0.83, bar_top = size * 0.20, bar_bottom = size * 0.80
     - bar_h = bar_bottom - bar_top, bar_w = max(6, size // 18)
     - Draw bar track (outline rect)
     - Map tilt [0, pi/2] to vertical position: tilt=0 (aligned with Y) = center of bar, tilt=pi/2 = bottom. indicator_y = bar_top + (tilt / (math.pi/2)) * bar_h
     - Draw indicator dot (radius 4) at (bar_cx, indicator_y), color (150, 160, 255)
     - Draw "Y" label above bar (font size 9 or size//14)
  k. W gauge (small circle in top-left corner of widget):
     - w_cx = size * 0.17, w_cy = size * 0.20, w_radius = max(7, size // 15)
     - Map w_align [-1, 1] to color: -1 = (80, 80, 220) blue, 0 = (140, 140, 140) neutral, +1 = (220, 80, 80) red. Interpolate linearly.
     - Draw filled circle then outline
     - Draw "W" label below circle (font size 9)
  l. Blit widget_surf to screen: screen.blit(widget_surf, (x, y))
